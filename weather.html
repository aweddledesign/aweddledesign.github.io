Weather App

I designed the weather app while I was learning to code in React. I found an open API through Dark Skies that allows you to input any longitude and latitude coordinates and find the weather information for that specific area. I designed this app to allow the user to search a location by either coordinates or the address. The current weather conditions are displayed using illustrated graphics. There are two alternate color schemes for day and night. The colors are dimmer for daytime and brighten as the sun goes down.
